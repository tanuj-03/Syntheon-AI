# Icons Directory

This directory contains icons for the Syntheon AI browser extension.

## Required Icons

- `icon16.png` - 16x16px icon for browser toolbar
- `icon48.png` - 48x48px icon for extension management
- `icon128.png` - 128x128px icon for Chrome Web Store

## Optional Platform Icons

- `google-meet.png` - Google Meet platform icon (20x20px)
- `zoom.png` - Zoom platform icon (20x20px)
- `teams.png` - Microsoft Teams platform icon (20x20px)

## Design Guidelines

- Use a consistent color scheme (blue/primary color)
- Include a recording/microphone element to indicate recording functionality
- Ensure good contrast and visibility
- Follow browser extension icon best practices

## Icon Specifications

- Format: PNG with transparency
- Color scheme: Blue (#3b82f6) primary, with white and gray accents
- Style: Modern, clean, professional
- Should work well on both light and dark browser themes
